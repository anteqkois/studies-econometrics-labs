2.3

Wybra ostateczny podzbir zmiennych objaniajcych (na 2 sposoby):

  • Wykorzysta w tym celu metod krokow wsteczn (bazujc na tecie istotnoci t-studenta). Przedstawi krtko ide tej metody.

W tej metodzie zaczynamy od penego modelu (wszystkie zmienne) i usuwamy najmniej istotne (te z najwikszym p-value w tecie t-Studenta), a zostan tylko istotne zmienne. Jeli p-value zmiennej w tecie t-Studenta jest wiksze ni 0.05, oznacza to, e zmienna nie jest istotna i powinna zosta usunita.

  • Wykorzysta kryteria informacyjne AIC, BIC.

Metoda opiera si na minimalizacji AIC i BIC. W skrcie: im nisze AIC/BIC, tym lepszy model. Wybieramy kolumn do usunicia na podstawie p-value. Jeli AIC/BIC si poprawiy (zmniejszyy), zostawiamy zmienn usunit i kontynuujemy proces. Jeli AIC/BIC si pogorszyy, cofamy usunicie i testujemy inn zmienn.

import statsmodels.api as sm
import numpy as np
import pandas as pd
from statsmodels.stats.anova import anova_lm

# Model podstawowy: tylko 'tax'
X_basic = df[['tax']]
X_basic = sm.add_constant(X_basic)  # Dodaje sta
y = df['price']

# Model rozszerzony: 'tax', 'mpg', 'engineSize'
X_extended = df[['tax', 'mpg', 'engineSize']]
X_extended = sm.add_constant(X_extended)

# Buduj modele
model_basic = sm.OLS(y, X_basic).fit()  # Estymacja modelu podstawowego
model_extended = sm.OLS(y, X_extended).fit()  # Estymacja modelu rozszerzonego

# Obliczam reszty
residuals_basic = model_basic.resid
residuals_extended = model_extended.resid

# Obliczam statystyk F dla testu Walda
# F-statystyka dla testu Wald to rnica w sumie kwadratw reszt (RSS) z dwch modeli
rss_basic = np.sum(residuals_basic ** 2)
rss_extended = np.sum(residuals_extended ** 2)
n = len(y)  # liczba obserwacji
k_basic = X_basic.shape[1]  # liczba zmiennych w modelu podstawowym
k_extended = X_extended.shape[1]  # liczba zmiennych w modelu rozszerzonym
m = k_extended - k_basic  # liczba dodanych zmiennych

# Statystyka F
F_statistic = ((rss_basic - rss_extended) / m) / (rss_extended / (n - k_extended))

# Obliczam warto krytyczn F (rozklad F-Snedecora)
from scipy.stats import f

# Stopnie swobody
df1 = m
df2 = n - k_extended
F_critical = f.ppf(0.95, df1, df2)

print(f"F-statystyka: {F_statistic}")
print(f"Warto krytyczna F: {F_critical}")

if F_statistic > F_critical:
    print("Odrzucamy hipotez zerow, zmienne objaniajce w modelu rozszerzonym s istotne.")
else:
    print("Nie odrzucamy hipotezy zerowej, zmienne objaniajce w modelu rozszerzonym nie s istotne.")

# Test Chowa (stabilno parametrw modelu)

# Szacuj model dla caego zbioru

# Dziel danyce na dwa
split_point = len(df) // 2
X1, X2 = X_train_2_3_a.iloc[:split_point, :], X_train_2_3_a.iloc[split_point:, :]

# Szacuj modele

model_2 = sm.OLS(y2, X2).fit()

RSS_1 = np.sum(model_1.resid ** 2)  # Resztowa suma kwadratw dla 1 przedziau
RSS_2 = np.sum(model_2.resid ** 2)  # Resztowa suma kwadratw dla 2 przedziau

alpha = 0.05

print(f"F={F_stat:.4f}, Warto krytyczna F={F_critical:.4f}")
